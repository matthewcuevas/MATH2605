TO RUN:
Navigate to the folder containing the .java files using a terminal and to compile the files, run:
    $ javac *.java

Now, to begin the program and start running individual methods, run:
    $ java Main.java


TO VIEW WRITTEN PORTIONS FOR EACH PROBLEM:
Navigate to the appropriate folder for each problem. That is, Problem 1 for Problem 1, Problem 2 for Problem 2, and Problem 3 for Problem 3.
