## To Run
#####On a Unix operating system

compile with
```sh
$ javac -classpath .:jama-1.0.3.jar JamaTest.java
```

Run with:
```sh
$ java -classpath .:jama-1.0.3.jar JamaTest
```

#####On Windows

compile with
```sh
$ javac -classpath .;jama-1.0.3.jar JamaTest.java
```

Run with
```sh
$ java -classpath .;jama-1.0.3.jar JamaTest
```

## Libraries used
